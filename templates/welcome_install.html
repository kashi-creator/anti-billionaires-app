One last thing

Put Sovereign on your home screen.

Welcome to the brotherhood. Before you do anything else, install Sovereign so it lives like an app on your phone — fullscreen, fast, and ready to ping you the moment something matters.

Install on iPhone

  1. Tap the Share button ↑ at the bottom of Safari.
  2. Scroll down and tap Add to Home Screen.
  3. Tap Add in the top right. The Sovereign icon will appear on your home screen.
  4. Open Sovereign from the home screen icon. Sign in once and you're set forever.

Install Sovereign

Tap install and confirm the prompt — the icon goes straight to your home screen.

Install on Desktop

  1. Look for the install icon in your browser's address bar (looks like a tiny computer with a downward arrow).
  2. Click it, then click Install in the prompt.
  3. Sovereign opens in its own dedicated window — like a native app.
